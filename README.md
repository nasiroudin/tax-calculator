# Tax Calculator App

## Description

A simple Tax Calculator that takes in a Postal Code and Annual Income to devise the calculated tax amount.
Each Postal Code has their own specific tax calculation namely:
| Postal Code | Tax Calculation Type |
|-------------|----------------------|
| 7441        | Progressive          |
| A100        | Flat Value           |
| 7000        | Flat Rate            |
| 1000        | Progressive          |
